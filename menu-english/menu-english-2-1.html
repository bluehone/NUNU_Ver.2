I-Box

I-Box is a product that combines the traditional bell system with motion-sensing system.
It consists of sensors that detect claims and movements that send and receive data in a large way.
Both the bell mode and the motion detection mode can be used at the same time. Depending on the situation, only one of the two can be selected.
In addition, real-time vehicle location information can be checked when connecting to the IOT, making it convenient for parents in kindergarten, hagwon, etc.

System Composition

Additional options provide IOT connectivity, and real-time vehicle location, accident status, etc. can be viewed via server and mobile when IOT is connected.
